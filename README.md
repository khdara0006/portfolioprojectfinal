## Technologies Used
- **Frontend**: React, CSS3
- **Backend**: Node.js, Express
- **Database**: MongoDB Atlas
- **Deployment**: AWS Amplify (Frontend), AWS Elastic Beanstalk (Backend)

## Features
- Responsive portfolio website
- Project management (CRUD operations)
- Contact form with MongoDB storage
- Admin dashboard

## Installation
1. Clone the repository
2. Install dependencies: `cd client && npm install` and `cd server && npm install`
3. Create `.env` files with your MongoDB URI
4. Run: `npm run dev` in both folders
